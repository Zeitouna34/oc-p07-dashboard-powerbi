# oc-p07-dashboard-powerbi
Dashboard Power BI multi-niveaux pour pilotage de portefeuille projets IT/Marketing chez Sanitoral. Suivi de 520 phases sur 4 régions avec alertes automatiques (écarts >15% coûts/délais/livrables). 3 vues décisionnaires (DG/DR/DP), cartographie mondiale, DAX avancé, Gantt interactif. Démontre maîtrise modélisation relationnelle &amp; storytelling data.


Résumé du projet :
Création d'un tableau de bord Power BI pour le suivi de performance de 520 phases de projets IT et Marketing chez Sanitoral (entreprise internationale soins bucco-dentaires). L'objectif était de fournir aux 3 niveaux de direction (DG, Directeurs Régionaux, Directeurs Pays) une vision 360° des projets avec alertes automatiques en cas de dépassement des seuils (>15% d'écart sur coûts, délais, livrables).
Tâches réalisées :
Collecte et nettoyage des données : Import fichiers CSV SIRH, traitement des incohérences, création du modèle relationnel (schéma en étoile).
Création de mesures DAX : Calculs d'écarts entre prévisionnel et réel, comptage automatique des projets en alerte, KPIs dynamiques par région/pays/type de projet.
Développement de 3 dashboards interactifs : Vue Directeur Général (carte mondiale, TOP 3 régions en alerte), Vue Directeur Régional (comparaisons pays), Vue Directeur Pays (détails projets avec diagramme de Gantt).
Présentation des résultats : Support PowerPoint avec méthodologie, choix de visualisations, et démonstration du Product Strategy Canvas utilisé pour le cadrage.
Compétences et outils mobilisés :
Outils : Power BI Desktop (visualisations, DAX), Power Query (nettoyage données), Excel (préparation).
Expertise : Modélisation relationnelle, création KPIs, data storytelling, conception multi-utilisateurs, respect RGPD.
Livrables :

Dashboard Power BI (.pbix) avec 3 vues décisionnaires
Product Strategy Canvas
Note de cadrage et dictionnaire des données
Support de présentation orale
